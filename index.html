Guy Barrette

Guy Barrette is a freelance Trainer/Developer based in Montreal, Canada and an Microsoft Azure MVP. His focus is to train organizations in using Azure thru Azure Academy and Docker/Kubernetes thru Kubernetes Academy, offering hands-on public and private courses. He has been a speaker at developers' conferences like Confoo, Microsoft TechDays and DevTeach. Guy was the leader of the Montreal .NET User Group for more than 23 years and is the Visual Studio Talk Show French podcast co-host. Guy has the following certifications: Azure Fundamentals, Azure Developer Associate and Azure Architect Design.
